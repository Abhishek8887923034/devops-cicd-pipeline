End-to-End CI/CD pipeline using Jenkins, Docker, and Kubernetes. Automates build and deployment process.
# 🚀 DevOps CI/CD Pipeline Project

## 📌 Description
This project demonstrates an end-to-end CI/CD pipeline using Jenkins, Docker, and Kubernetes.

## 🛠️ Tools Used
- Jenkins (CI/CD)
- Docker (Containerization)
- Kubernetes (Deployment)
- AWS (Cloud - optional)

## ⚙️ Workflow
1. Code pushed to GitHub
2. Jenkins builds Docker image
3. Container runs application
4. (Optional) Deploy to Kubernetes

## 📷 Output
Simple web app running via Docker container
